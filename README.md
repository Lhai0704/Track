## 项目架构

状态管理框架是Provider，整个项目架构分为三层：

- View层：展示页面
- Model层：处理数据
- Logic层：逻辑操作

| 目录     | 备注             |
| -------- | ---------------- |
| config   | 配置             |
| data     | 数据             |
| database | 数据库操作       |
| items    | List列表的item类 |
| logic    | Logic层          |
| model    | Model层          |
| pages    | View层           |
| util     | 工具类           |
| Widgets  | 封装widget       |
